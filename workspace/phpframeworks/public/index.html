However, if you are seeing this page, it means you are using IP Address, not virtual host!

This is probably not the page you want.

This page is being served from /var/www/html/index.html, and is the default vhost (virtual host) that comes with most web servers.

What you want to do is to add all vhosts to your hosts file.

On OSX and Linux, this is located at /etc/hosts, on Windows it is located at C:\Windows\System32\drivers\etc\hosts.

Add a line with the IP address of this server and all vhosts you chose. For example, if the IP address is 192.168.56.101, and you chose vhost awesome.dev, you will want to enter the following: 

192.168.56.101  local.php.frameworks
192.168.56.101  phpframeworks.dev   php.frameworks.dev
192.168.56.101  cakephp.dev         www.cakephp.dev
192.168.56.101  kohana.dev          www.kohana.dev
192.168.56.101  laravel.dev         www.laravel.dev
192.168.56.101  phalconphp.dev      www.phalconphp.dev
192.168.56.101  symfony.dev         www.symfony.dev
192.168.56.101  zend.dev            www.zend.dev

If you forgot the IP address you chose, check the config file at puphpet/config.yaml.
